Deep vessel segmentation system

It's a project of my computer graphic course. The software can offer various ways for segmentation of blood vessels in a retinal image and display them.

This project implements a software which can generate both 2D and 3D images of the original input retinal images by level set propagation with local phase information to capture the boundaries of vessels.
